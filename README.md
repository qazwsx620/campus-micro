# 校园微旅

## 项目定位
面向新生/游客的7页响应式校园导览，支持搜索、收藏、夜间模式。

## 技术栈
HTML5语义化 + CSS3(Grid/Flex) + JS(es6) + Canvas + Bootstrap5 + GitHub Pages

## 核心功能
1. 首页轮播+Canvas地图点击跳转
2. 地标列表实时搜索+高亮
3. 详情页图片放大镜+收藏♥+打分+评论        
4. 本地存储收藏夹+一键清空
5. 夜间模式+三端响应式
