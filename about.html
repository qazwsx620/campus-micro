项目介绍

「校园微旅」是面向新生、游客的可视化导览平台，涵盖 8 处地标、收藏、订阅、夜间模式等完整功能，采用 HTML5+CSS3+JS+Bootstrap5 全栈技术栈，已部署至 GitHub Pages。

开发时间线

  • Day1 首页 + Canvas 平面图
  • Day2 地标列表 + 搜索高亮
  • Day3 详情页 + 收藏功能
  • Day4 收藏夹 + 订阅 + 关于
  • Day5 部署上线 + 文档录屏

技术栈

HTML5 语义化 CSS3 Flex/Grid JavaScript ES6 Canvas 绘图 Bootstrap5 GitHub Pages
